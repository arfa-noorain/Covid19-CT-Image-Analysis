# Chest Computed Tomography Analysis for Probabilistic Determination of Covid‑19

## Project Overview
This technical project proposes a model using modern image processing techniques on chest CT scans to assist in the probabilistic determination of Covid-19 infections. The system was designed to analyze visual data patterns to assist medical and technical workflows.

## Technical Stack & Environment
* **Language:** Python
* **Platform:** Anaconda Environment
* **Domain:** Image Processing / Computational Analysis

## Publication
This research was documented and published in the **SCOPUS Journal: Psychology and Education (2021)** under the title *"Survey on Probabilistic Determination of Corona Virus Infection"*.
